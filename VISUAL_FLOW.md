## ডাটা ফ্লো:

```
User Input
    ↓
Validation
    ↓
API Request (POST /api/auth/login)
    ↓
Backend Processing
    ├─ Email Check
    ├─ Password Verification
    └─ JWT Token Generation
    ↓
Response
    ├─ Success: {token, user}
    └─ Error: {message}
    ↓
Frontend Handling
    ├─ Success: Show Popup → Redirect
    └─ Error: Show Error Popup
    ↓
User Action
    ├─ Success: Dashboard
    └─ Error: Retry Login
```
